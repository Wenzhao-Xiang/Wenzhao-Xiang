Detailed Description

Performs Cholesky decomposition of matrix \(A = L*L^T\) and solves matrix equation \(A*X=B\).

Parameters
src1pointer to input matrix \(A\) stored in row major order. After finish of work src1 contains lower triangular matrix \(L\).
src1_stepnumber of bytes between two consequent rows of matrix \(A\).
msize of square matrix \(A\).
src2pointer to \(M\times N\) matrix \(B\) which is the right-hand side of system \(A*X=B\). B stored in row major order. If src2 is null pointer only Cholesky decomposition will be performed. After finish of work src2 contains solution \(X\) of system \(A*X=B\).
src2_stepnumber of bytes between two consequent rows of matrix \(B\).
nnumber of right-hand vectors in \(M\times N\) matrix \(B\).
infoindicates success of decomposition. If *info is false decomposition failed.
