Detailed Description

Bitwise AND: dst[i] = src1[i] & src2[i]
Bitwise OR: dst[i] = src1[i] | src2[i]
Bitwise XOR: dst[i] = src1[i] ^ src2[i]
Bitwise NOT: dst[i] = !src[i]
